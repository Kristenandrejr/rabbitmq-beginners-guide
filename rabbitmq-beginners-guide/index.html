This simple project shows how to send and receive messages using Python and RabbitMQ.

1. Requirements

  • Install RabbitMQ on your system.
  • Install the pika Python library using:
    pip install pika

2. How it works

The Producer sends a message to a queue called hello. The Consumer listens to the same queue and prints any message it receives.

3. How to run

  1. Open two terminal windows.
  2. In the first, run: python consumer.py
  3. In the second, run: python producer.py
  4. Watch your consumer print the received message!
